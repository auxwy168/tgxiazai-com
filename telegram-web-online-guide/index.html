为什么要用 Telegram Web 网页版？

你可能遇到过这种情况：临时借用别人的电脑、在学校机房上机、出差时用酒店大堂的公用电脑，又或者你的手机刚好没电了，但你需要马上回复一条 TG 消息。装客户端？来不及。手机验证码？手机都没电哪来的验证码。

这时候 Telegram Web 网页版就是救命稻草。

打开浏览器，输入 https://web.telegram.org/，扫码或输入手机号，3 秒内就能开始聊天。不用下载安装包，不占硬盘空间，不留聊天记录（如果你勾选了隐私模式），用完关掉浏览器窗口，干干净净。

Telegram Web 有两个版本？K 版 vs Z 版

直接说重点：Telegram 官方提供了两个网页版，它们的目标用户、界面、底层技术都不一样。

K 版（版本号 “K”）— 经典轻量版

访问地址：https://web.telegram.org/k/

K 版是 Telegram 在 2021 年推出的完全重写的网页版，底层基于 WebAssembly 和 Web Workers，运行效率比老版本高得多。全称 “Telegram Web K”，采用了 Telega 库实现 MTProto 协议的浏览器端加密。

K 版的特点：

  • 界面和桌面版几乎一样，左侧对话列表 + 右侧聊天窗口的双栏布局
  • 支持文件夹（Chat Folders）功能，可以按工作、朋友、群组分类
  • 支持频道浏览、创建和管理的完整功能
  • 支持贴纸、GIF、表情回复（Emoji Reactions）
  • 不支持语音通话和视频通话
  • 消息的端到端加密直接在浏览器内完成，不需要经过服务器中转解密
  • 可以独立缓存消息，断网后也能查看历史记录
💡 小提示：K 版目前是 Telegram 官方推荐的网页版方案。如果你希望获得最接近桌面客户端的网页体验，直接用 K 版就行。

Z 版（版本号 “Z”）— 移动端风格版

访问地址：https://web.telegram.org/z/

Z 版是更早期的版本，界面更偏向手机端的单栏设计。虽然现在官方已经把重心转向 K 版，但 Z 版仍然可用。

Z 版的特点：

  • 单栏布局，适合窄屏（像在手机 App 里操作）
  • 加载速度极快，对老旧浏览器兼容性好
  • 功能比较基础：发消息、发图片、建群、频道浏览都可以
  • 不支持文件夹（Chat Folders）
  • 不支持贴纸商店浏览
  • 更适合配置低的电脑或旧版浏览器使用

两个版本的核心差异速查

特性K 版Z 版
界面布局双栏（桌面风格）单栏（手机风格）
文件夹功能✅❌
语音/视频通话❌❌
贴纸商店✅❌
离线消息缓存✅有限
运行性能高（WebAssembly）一般
官方推荐是维护模式

一句话总结：日常用 K 版，旧电脑/旧浏览器用 Z 版。

怎么登录 Telegram Web 网页版？

Telegram 网页版支持两种登录方式，推荐用扫码登录，最快也最安全。

方式一：二维码扫码登录（推荐）

这是最快的登录方式，前提是你的手机端 Telegram 已经登录了。

  1. 打开浏览器，访问 https://web.telegram.org/k/
  2. 页面中央会出现一个二维码
  3. 打开你手机上的 Telegram App
  4. 点击底部 「设置」（Settings）→ 选择 「设备」（Devices）
  5. 点击右上角的 「连接设备」（Link Desktop Device）
  6. 把手机摄像头对准电脑屏幕上的二维码
  7. 手机端会弹出确认提示，点击 「确认」（Confirm）
  8. 网页版自动跳转进入聊天界面，登录完成

整个过程大约 10 秒，不需要输入手机号，不需要验证码。对于已经登录了手机端 TG 的用户来说，这是最快的方式。

方式二：手机号 + 验证码登录

如果你手机不在身边，但记得自己的 TG 账号绑定的手机号，可以用这种方式。

  1. 访问 https://web.telegram.org/k/
  2. 点击二维码下方的 「Log in by phone number」
  3. 选择国家/地区为 China (+86)
  4. 输入你的手机号，点击 Next
  5. 系统会向该手机号发送验证码（App 内推送或短信）
  6. 输入收到的 5 位验证码
  7. 如果设置了二次验证密码（Two-Step Verification），输入密码
  8. 登录成功
⚠️ 安全警告：在公共电脑上不要用手机号+验证码方式登录，因为登录后这台电脑就会被添加到你的 Telegram 设备列表里。强烈建议只在个人电脑上用这种方式。公共电脑只用扫码登录，并且用完记得退出。

二维码扫不出来怎么办？

  • 确认手机和电脑的网络都可以访问 Telegram 服务
  • 尝试刷新网页，重新生成二维码
  • 确认手机端 TG 已经正常登录（不是被封号或掉线状态）
  • 尝试用 K 版而不是 Z 版（K 版的二维码兼容性更好）
  • 如果持续失败，可能是当前网络环境限制了 Telegram 的连接，参考代理设置章节

网页版 vs 桌面版：功能差在哪里？

Telegram 网页版很方便，但它不是桌面客户端的完全替代品。搞清楚差在哪里，你就知道什么时候该用网页版，什么时候该乖乖装客户端。

视频通话❌ 不支持✅ 支持
群组语音/视频通话❌ 不支持✅ 支持

核心差异在通话和加密私聊。如果你平时只用 TG 发消息、浏览频道、群聊吹水，网页版完全够用。但如果你需要打语音或视频电话，或者要用 Secret Chat 功能，那就必须装手机 App 或桌面客户端了。

网页版的独特优势

反过来，网页版也有一些客户端没有的优势：

  • 零安装：不占磁盘空间，不需要管理员权限安装程序
  • 跨平台一致性：Windows、Mac、Linux、ChromeOS，只要有浏览器就能用
  • 隐私残留少：关闭浏览器窗口，本地聊天缓存不会长时间保留（尤其隐私模式）
  • 多开方便：不同浏览器、不同配置文件可以同时登录不同账号，比客户端切号方便
  • 企业/学校环境友好：不少公司电脑不能随便装软件，但浏览器人人都有

网页版怎么设置中文？

Telegram 官方网页版默认只有英文界面。不过通过官方语言包机制，60 秒就能切到中文。

K 版中文设置步骤

  1. 在 K 版界面中，点击左上角的三条横线（汉堡菜单）
  2. 选择 「Settings」（设置）
  3. 在设置页面找到 「Language」 选项
  4. 在搜索框输入 「Chinese」
  5. 会出现多个中文语言包，推荐选择 「简体中文（@zxcvbnm2112）」，这是目前社区维护最活跃的中文包
  6. 点击该语言包，系统会自动下载并应用
  7. 界面立刻变成中文，无需重启

Z 版中文设置步骤

  1. 点击左侧的 「Settings」（设置）
  2. 找到 「Language」 选项
  3. 同样搜索 Chinese 选择中文包即可
💡 注意：网页版的语言包和桌面版是独立的，你在网页版设置的中文不会自动同步到手机或电脑客户端。每个平台需要单独设置。如果语言包加载后个别地方还是英文，刷新页面即可。

公共电脑使用网页版的安全指南

在图书馆、网吧、酒店商务中心用 TG 网页版？下面这几条必须做到，缺一条都可能让你的账号出事。

1. 优先使用浏览器的隐私/无痕模式

  • Chrome/Edge：按 Ctrl + Shift + N 打开无痕窗口
  • Firefox：按 Ctrl + Shift + P 打开隐私窗口

无痕模式关闭窗口后自动清除所有缓存、登录状态和 cookie，不会有任何残留。

2. 绝对记住退出登录

聊完天后，点击左上角菜单 → 「Settings」→ 最底部的 「Log Out」。不要只关浏览器窗口 —— 浏览器可能记住了你的登录状态，下一个人打开就能直接看你的聊天记录。

退出后再检查一遍：重新打开 https://web.telegram.org/k/，确认显示的是登录页面而不是直接进入聊天界面。

3. 在手机端检查并移除陌生设备

登录网页版后，这台电脑就被加入了你账号的「活跃设备」列表。用完退出登录后：

  1. 打开手机 TG → 「设置」→ 「设备」
  2. 查看所有登录过的设备
  3. 如果那台公共电脑还显示在列表里（偶尔退出登录后不会自动消失），手动点进去选择 「Terminate Session」
  4. 确认设备列表里只有你自己的手机、电脑、平板

4. 不要在公共电脑上传输敏感文件

网页版在公共电脑上没有端到端加密保障（Secret Chat 在网页版不可用），传输的文件可能被浏览器的缓存机制暂存在本机。涉及身份证照片、银行卡信息、密码等敏感内容，绝不能通过公共电脑的网页版发送。

5. 检查周围环境

输入密码或验证码时，确认身后没有人盯着你的屏幕。公共场合的肩窥（Shoulder Surfing）是最常见的信息泄露方式之一。

⚠️ 最重要的提醒：如果你在公共电脑上登录了 Telegram 网页版，离开时没退出登录，下一个使用这台电脑的人可以直接看到你所有的聊天记录、群组列表和联系人信息。这不是危言耸听——现实中发生过不止一次。请在离开前务必确认退出登录并关闭浏览器。

网页版网络连接问题排查

在中国大陆使用 Telegram 网页版，最常见的问题是页面打不开、二维码刷不出来、消息一直「Connecting」。

原因分析

Telegram 网页版需要连接到 Telegram 的服务器来获取数据和同步消息。如果网络层面无法访问 Telegram 服务器，网页版就会一直转圈。

解决方案

  1. 检查网络环境：确认当前网络可以访问 Telegram 服务。如果连 https://web.telegram.org/ 都打不开，说明是网络层面的问题。
  2. 使用代理或 VPN：配置浏览器代理或者使用系统级 VPN。K 版网页版会自动使用浏览器的代理设置。
  3. 尝试不同的浏览器：Chrome、Edge、Firefox 的代理配置方式不同，有的浏览器插件可以更方便地配置代理规则。
  4. K 版 vs Z 版切换：有时候某个版本的 CDN 节点在当前网络下不通，换另一个版本可能就好了。

关于网络方面的详细方案，请参考网站内的代理网络分类文章，里面有不同场景的配置方法。

常见问题 FAQ

Telegram 网页版和 App 共用同一个账号吗？▼
是的。Telegram 所有平台（手机、平板、电脑、网页）都共用一个账号，用同一个手机号登录。消息会自动在所有设备间同步，你在网页版发的消息在手机上也能看到。这是 Telegram 的云同步架构，和微信的「手机为主、电脑为辅」逻辑不同。
网页版可以同时登录多个 TG 账号吗？▼
单个浏览器窗口只能登录一个账号。但你可以通过以下方式实现多开：① 用不同的浏览器（Chrome 登一个、Edge 登一个、Firefox 登一个）；② 用浏览器的多用户/多配置文件功能（Chrome 的 Profile 功能）；③ 正常窗口登录一个，无痕窗口再登录另一个。注意无痕窗口关闭后需要重新登录。
Telegram 网页版安全吗？会被监控吗？▼
从技术层面看，Telegram K 版网页版使用 MTProto 协议在浏览器端完成加密，数据在传输过程中是加密的。但有两个前提：① 你连接的是正版 `https://web.telegram.org/`，不是钓鱼网站；② 你使用的电脑本身没有被安装键盘记录、截屏等恶意软件。在公共电脑上，最大的安全风险不是传输过程，而是本地设备本身，所以务必退出登录并清空缓存，详见本文安全指南章节。
网页版能发文件和图片吗？文件大小有限制吗？▼
可以。网页版支持发送图片、视频、文档、语音消息等几乎所有文件类型。文件大小限制和客户端一致：普通用户单文件最大 2GB，Telegram Premium 用户最大 4GB。发送方式支持拖拽到聊天窗口或点击附件按钮选择文件，和桌面版操作方式一样。
为什么网页版一直显示「Connecting」连不上？▼
「Connecting」说明浏览器无法连接到 Telegram 的服务器。常见原因：① 网络层面无法访问 Telegram 服务（最常见）；② 浏览器代理设置问题；③ Telegram 服务端临时故障（极少见）。解决方法：检查网络连接和代理配置，尝试换 K/Z 版，或换一个网络环境试试。
K 版和 Z 版能同时登录吗？▼
技术上可以，但不推荐。同一个 TG 账号可以在两个不同的网页版本中同时登录，因为它们被视为两个不同的会话（Session）。但这样做会导致消息通知混乱，两个窗口都会弹出新消息提醒，而且你无法确定自己在哪个窗口回复的。除非有特殊需求（比如一个窗口用来看频道、另一个用来聊天），否则不要同时打开两个网页版。
